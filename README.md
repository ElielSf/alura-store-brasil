# Desafio Alura Store Brasil

Este projeto tem como objetivo analisar o desempenho de quatro lojas pertencentes ao Senhor João,
a fim de identificar qual unidade apresenta os piores resultados gerais e deve ser considerada para venda.

## Indicadores analisados
- Faturamento total
- Categorias de produtos mais e menos vendidas
- Média das avaliações dos clientes
- Produtos mais e menos vendidos
- Custo médio de frete

## Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Estrutura do projeto
- `AluraStoreBrasil.ipynb`: notebook contendo a análise exploratória, visualizações e relatório final.

## Observação
Os dados utilizados na análise não estão disponíveis neste repositório por questões de tamanho e privacidade.
